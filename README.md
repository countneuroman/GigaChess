# GigaChess

Проект шахмат с донатными функциями.

Вайбкожу во все тяжкие, сам код не пишу, тренируемся с ИИ.
На данный момент использую Claude Code + Beads из инструментов, все в VSCode.

## Стек технологий

| Слой           | Технологии                                      |
|----------------|--------------------------------------------------|
| **Frontend**   | React 19, TypeScript, Chessground, Parcel, LESS  |
| **Backend**    | .NET 9, ASP.NET Core, OpenAPI                    |
| **Инфраструктура** | Docker, Docker Compose, Nginx (reverse proxy) |
| **Observability**  | OpenTelemetry, Jaeger                         |




## Запуск

```bash
docker-compose up --build
```

| Сервис    | URL                        |
|-----------|----------------------------|
| Frontend  | http://localhost:8080       |
| API       | http://localhost:7130       |
| Jaeger UI | http://localhost:16686      |

## Куда движется проект

1. **Полноценный шахматный движок** — модели фигур и ходов уже на месте, следующий шаг — реализация валидации ходов, определение легальных ходов, шаха и мата
2. **Игровой цикл** — управление состоянием партии: очерёдность ходов, взятие фигур, специальные ходы (рокировка, взятие на проходе, превращение пешки)
3. **Мультиплеер в реальном времени** — Nginx-шлюз и контейнерная архитектура готовы к масштабированию; логичное продолжение — WebSocket для синхронизации партий между игроками, хранение состояния игроков.
4. **Улучшение фронтенда для удобства игроков** - сделать симпатичную морду где поиммо шахматной доски будет удобная информация о партии
5. **Внедрение донатных функций** — технически интересные premium-фичи без влияния на исход партии:
   - Премиум-аналитика: движковый разбор партий после завершения (интеграция со Stockfish/ONNX), граф ошибок, оценка позиции по ходам
   - Персональный тренер на базе LLM: объяснение ошибок человеческим языком, рекомендации по дебютам на основе истории партий
   - Расширенная статистика: паттерны игры, heatmap активности фигур, слабые стороны по фазам партии (дебют/миттельшпиль/эндшпиль)
   - Облачная база партий с полнотекстовым поиском по позициям (bitboard-индексация)
   - Кастомные турниры: создание приватных турниров с настройкой контроля времени, формата (Swiss/Round Robin) и правил
